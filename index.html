This project is inspired by https://www.covidvisualizer.com/ as I
wanted to try to recreate it using d3.js.

The data is fetched from https://covid19api.com/ which
is updated a couple of times a day.

Some of the data is a bit weird, for example it says Sweden has a lot more active cases than it actually does.
